# 🔐 Py Port Scanner (Python GUI)

A multithreaded network port scanner built using Python with a simple Tkinter GUI. This tool allows users to scan a target IP address or hostname, detect open ports, and view results in real time.

---

## 🚀 Features

- Scan any IP address or hostname  
- Fast multithreaded scanning  
- Real-time progress tracking  
- Detects open TCP ports  
- Displays common service names  
- Save scan results to file  
- Simple and beginner-friendly GUI  

---

## 🛠️ Technologies Used

- Python  
- Socket Module  
- Threading  
- Tkinter (GUI)  
- Queue Module  
- Time Module  
- ChatGPT (for guidance & documentation)  




---

## ▶️ Usage

1. Enter Target IP / Hostname  
2. Set Start Port and End Port  
3. Click Start Scan  
4. View results in real time  
5. Save results if needed
